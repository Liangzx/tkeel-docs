API列表

平台 API 分为管理平台以及用户平台。

管理平台​

管理平台包括以下API：

  • 平台基础API: 包含平台配置接口、前端菜单接口
    • 平台配置接口支持了平台样式服务以及基础信息
  • 系统管理：包含平台管理员 Token 颁发、Plugin 白名单、Token 验证接口、Admin重设密码接口
  • 插件管理：包含插件列表、插件启用、停用、注册、更新、插件仓库管理
  • 租户管理：包含租户空间管理、单点登录配置
  • 计量服务：包含用量数据接口
  • 登录认证：包含登录认证

租户平台​

租户平台包括以下API：

  • 租户API：包含用户管理、权限管理
  • 设备API：包含设备服务及映射服务、设备组服务、设备模板服务
  • 数据服务API：包含数据查询、订阅管理、数据路由、前端Websocket订阅
